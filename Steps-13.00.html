Adding project logo's, Title and Background 13

Adding a background:

1.     Click anywhere on the Canvas

2.     Go to Format Report Page then Canvas Background

3.     Select Image from the downloaded file

4.     Change Transparency = 0%

Adding Colaberry Logo:  

1.     Insert: Image and select Colaberry logo from the downloaded file

2.     Go to General tab and set Properties: Size : Height = 102, Width = 261

                                                Position: Horizontal = 0, Vertical = 0

Adding Suicide Logo:

1.     Insert: Image and Select Suicide Logo from the downloaded file and change Image Style = FILL

2.     Go to General tab and set Property: Size: Height = 102, Wedth = 112

                                                Position: Horizontal = 1167, Vertical = 0

Adding Title:

1.     Insert a Text Box and insert ”Global Suicide Trends: Analyzing Demographics and Generational Patterns.(1985 - 2015)”

2.     The Font and style will be set to Segoe UI , 24

3.     Go to General tab and set Property: Size: Height = 102, Width = 825

                                                Position: Horizontal = 315, Vertical = 0
