Adding Map Scale:

1.     Go to Insert tab and Add Image

2.     Select “MapScale” from the file you downloaded

3.     Go to format tab and general then Properties size = 25, Width = 145, Position Horizontal = 776, Vertical = 344

Adding Map Scale Legend:

1.     Go to Insert tab ad Add Text box

2.     Insert “High      Low” text

3.     Go to format tab and general then Properties size = 27, Width = 112, Position Horizontal = 793, Vertical = 360
