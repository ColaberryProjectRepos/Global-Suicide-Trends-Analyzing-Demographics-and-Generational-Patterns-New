Transforming the data in power Query Editor 9


In this step we will clean and transform the data

1.     Click on your table name and select add column.

2.     Add a new column “Modified date”.

3.     Click on your table name one more time and select edit query.

4.     Select the country column then from the top menu select split.

5.     Name the country and continent columns.

6.     Select the Year column and unselect 2016.

7.     Save and continue.
